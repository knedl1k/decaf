\section{Scalable Training Architecture}
Training a deep metric learning model on a dataset with extreme cardinality (e.g., over $10^5$ unique classes) introduces significant computational bottlenecks. Specifically, the weight matrix $\mathbf{W}$ in the ArcFace head requires substantial VRAM, scaling linearly with the number of unique cards. To address this, the training pipeline was engineered for high-throughput distributed execution.

\subsection{Distributed Data Parallelism and Synchronization}
The model optimization is parallelized across multiple accelerators using PyTorch's~\cite{paszke2019pytorch} \ac{DDP}~\cite{ddppytorch}. Unlike standard \texttt{DataParallel}\footnote{\url{https://docs.pytorch.org/docs/stable/generated/torch.nn.DataParallel.html}}, which suffers from \ac{GIL} contention and uneven memory distribution~\cite{pytorch_ddp}, \ac{DDP} spawns independent processes for each device. To maintain mathematical consistency of the batch statistics across distributed workers, standard Batch Normalization layers were replaced with Synchronized Batch Normalization (\texttt{SyncBatchNorm}\footnote{\url{https://docs.pytorch.org/docs/stable/generated/torch.nn.SyncBatchNorm.html}}). This ensures that the variance and mean are computed globally, preventing individual GPUs with small effective batch sizes from destabilizing the embedding space.

\subsection{Mixed Precision Training} % https://docs.pytorch.org/tutorials/recipes/recipes/amp_recipe.html
To further optimize memory footprint and memory bandwidth, \ii{mixed precision} was integrated into the training loop. The forward pass and loss computation are executed in \texttt{bfloat16} (Brain Floating Point)~\cite{bfloat}, a format that preserves the dynamic range of 32-bit floats while halving the memory requirement. The weights and optimizer states remain in \texttt{float32} to prevent precision degradation during gradient updates. This approach effectively doubles the maximum allowable batch size, leading to smoother gradient estimates and faster convergence.
