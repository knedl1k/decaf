\chapter{Attachments}\label{app:A}

The archives attached to this thesis contains the complete source code, datasets, trained models, and hardware design files necessary to fully reproduce and deploy the proposed robotic platform. The root directory of the archive is structured as follows:

\begin{itemize}
    \item \texttt{src}: Contains the complete software implementation, including the PyTorch training pipeline, evaluation scripts, ONNX edge inference runtime and the Arduino firmware for low-level hardware control.
    \item \texttt{real\_dataset}: Contains the labeled validation dataset consisting of real-world Magic: The Gathering card photographs captured under deployment conditions, used for benchmarking.
    \item \texttt{trainedNN}: Contains the trained neural network checkpoint and card database.
    \item \texttt{STL}: Contains the \texttt{.STL} files for all custom mechanical and structural components.
\end{itemize}

Everything is also available at \url{https://github.com/knedl1k/decaf}.
