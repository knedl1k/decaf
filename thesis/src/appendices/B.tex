\chapter{Computational Environment}\label{app:B}
All computational experiments, including model training, index building, and inference, were executed on the Research Center for Informatics\footnote{\url{https://rci.cvut.cz/}} cluster. To ensure exact reproducibility and proper hardware utilization across the Slurm workload manager, the software environment was standardized using the environment modules system.

The following modules were loaded for all execution scripts:
\begin{itemize}
    \item \texttt{Python/3.13.5-GCCcore-14.3.0}
    \item \texttt{PyTorch/2.10.0-foss-2025b-CUDA-12.9.1}
    \item \texttt{Albumentations/2.0.8-foss-2025b-CUDA-12.9.1}
    \item \texttt{OpenCV/4.12.0-foss-2025b-CUDA-12.9.1-contrib}
    \item \texttt{timm/1.0.25-foss-2025b-CUDA-12.9.1}
\end{itemize}

The pretrained model weight of \ac{ResNet}-50 used in this thesis is available at \url{https://download.pytorch.org/models/resnet50-0676ba61.pth}.
